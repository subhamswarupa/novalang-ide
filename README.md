# 🚀 NovaLang IDE

A modern, browser-based Integrated Development Environment (IDE) built for the **NovaLang** programming language. NovaLang IDE provides an intuitive coding experience with syntax highlighting, code editing, and program execution in a clean and responsive interface.

🌐 **Live Demo:** https://novalang-ide.vercel.app/

---

## 📌 Features

- ✨ Modern code editor powered by Monaco Editor
- 🎨 Syntax highlighting
- ⚡ Fast and responsive UI
- 📂 Built-in example programs
- ▶️ Run NovaLang programs directly in the browser
- 💻 Cross-platform and web-based
- 📱 Responsive design

---

## 🛠️ Tech Stack

- React 19
- TypeScript
- Vite
- Monaco Editor
- Tailwind CSS
- Framer Motion
- Lucide React
- Vercel (Deployment)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/subhamswarupa/novalang-ide.git
```

Move into the project directory

```bash
cd novalang
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
